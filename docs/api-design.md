# 초기 API 설계 후 주요 변경점

### 인증(Auth) 기능 분리
사용자 관리(User) 범주에 있었던 회원가입, 로그인, 로그아웃을
인증 기능으로 분리하고, 토큰 갱신을 추가했습니다.
사용자 관리의 정보 수정에서 비밀번호를 변경하는 기능을 따로 분리했습니다.

### 장바구니(Cart) 기능 확장
장바구니에 담은 도서의 수량을 변경하는 기능을 추가했습니다.

### 주문(Order) 기능 확장
주문 조회 기능에서 단일 주문의 상세 조회와 목록 조회 기능을 분리해
2개의 엔드포인트로 구성했습니다.
관리자 전용 기능인 주문 상태 관리 기능을 추가했습니다.
